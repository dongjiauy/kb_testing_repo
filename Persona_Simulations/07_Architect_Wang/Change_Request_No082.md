# 施工图设计变更通知单 (No. 082)
**项目**: 滨江金融中心二期
**日期**: 2026-03-09
**专业**: 建筑、结构
**事由**: 地下室顶板覆土厚度调整

## 1. 变更背景
- 甲方要求: 原设计地下室顶板覆土厚度为1.2m，现甲方因园林景观调整，需增加大乔木种植，要求覆土厚度增至1.8m。

## 2. 变更内容
- **建筑专业**: 调整地下室顶板找坡层及排水沟深度。
- **结构专业**: 
  - 顶板恒载增加 (1.8m x 18kN/m3 = 32.4kN/m2)。
  - 原配筋 (HRB400 Φ20@150) **已不足**。
  - 需重新计算并加强配筋至 (HRB400 Φ25@150)，梁截面由 400x800 加大至 500x900。

## 3. 注意事项
- 增加造价约 ¥200/m2。
- 需复核柱子承载力。
- 本变更需重新送审图公司复审。

**签发人**: 王工 (注册建筑师)
**审核人**: 老张 (总工程师)
