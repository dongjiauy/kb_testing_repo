# 政府沟通纪要: 关于 L4 自动驾驶路测牌照申请
**日期**: 2026-03-10
**对象**: 北京市交通委 (王主任)
**事由**: Robotaxi 开放道路测试申请

## 1. 核心诉求
- **目标**: 申请亦庄全域开放测试（包括晚高峰时段）。
- **理由**: 目前 1.2M 公里测试里程，0 重大事故。

## 2. 政府反馈 (困难点)
- **安全**: 近期某竞品发生一起追尾事故，政府现在非常谨慎，要求提供更严格的接管数据。
- **数据**: 要求所有路测数据必须接入政府监管平台（实时）。
- **舆论**: 不希望因为测试车影响早晚高峰交通拥堵。

## 3. 应对策略
- **数据脱敏**: 上传数据时去掉高精地图敏感信息（需技术支持）。
- **错峰**: 主动提出只在非高峰时段进行测试（10:00-16:00, 20:00-06:00）。
- **保险**: 增加每辆车的商业险保额至 500 万。

## 4. 行动项
- **李总**: 下周亲自带队去交通委汇报，带上最新的安全接管报告。
- **Kelly**: 准备好合规材料，特别是数据安全审计报告。
