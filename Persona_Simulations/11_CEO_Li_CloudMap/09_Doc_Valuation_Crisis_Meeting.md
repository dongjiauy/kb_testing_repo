# 关于 C 轮估值调整的紧急会议
**日期**: 2026-03-14
**参与人**: 李总 (CEO), Kelly (CFO)

## 1. 估值现状
- **上一轮**: B 轮投后估值 $8 亿美金。
- **目标**: C 轮原计划 $10 亿美金。
- **Mike (Sequoia)**: 要求砍 20%，即 C 轮估值 **$6.4 亿美金**。

## 2. 影响分析
- **老股稀释**: 创始团队股份将大幅稀释，可能失去绝对控制权。
- **反稀释条款 (Anti-Dilution)**: B 轮投资人会触发棘轮条款，需要补偿股份。
- **团队期权**: 员工期权池缩水，可能引发离职潮。

## 3. 应对方案
- **接受砍价**: 
  - *优点*: 快速拿到救命钱，活下去。
  - *缺点*: 创始团队利益受损严重。
- **硬抗**: 
  - *优点*: 赌一把顺丰意向书，用 5000 万合同抬估值。
  - *缺点*: 如果顺丰不签，直接断粮。
- **找新投资人**: 
  - *优点*: 找产业资本 (如车企)，可能不砍估值。
  - *缺点*: 时间来不及，尽调至少 3 个月。

## 4. 结论
- **李总**: 必须拿下顺丰意向书！这是唯一的筹码。如果有了 500 台订单，我就敢跟 Mike 谈平轮 ($8 亿)。
- **Action**: 下周飞深圳，把合同敲定。
